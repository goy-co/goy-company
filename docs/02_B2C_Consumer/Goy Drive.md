# Goy Drive // Soberania de Dados e Armazenamento

O **Goy Drive** é o serviço que transforma o ecossistema **[[The Goy Company]]** numa ferramenta de produtividade e soberania digital definitiva. Enquanto as Big Tech mineram os teus ficheiros, a Goy Co. oferece o "armazém", mas apenas o utilizador detém a chave da porta.

## 1. O Produto: "Privacy by Physics"
Segurança absoluta através de protocolos descentralizados:
*   **Encriptação Zero-Knowledge:** Os ficheiros são cifrados localmente antes do upload. A Goy Company armazena apenas "ruído" digital ilegível.
*   **Nostree Layout:** Metadados (nomes, pastas, datas) são cifrados e guardados como eventos Nostr privados.
*   **Interoperabilidade:** O utilizador é livre de migrar os seus dados para outro fornecedor, bastando apontar a sua chave para outro relay compatível.

## 2. Integração com o Ecossistema Goy
O Goy Drive atua como o disco rígido universal das nossas apps:
*   *[[Goy Chat]]:** Repositório central de media e ficheiros partilhados em conversas.
*   *[[Goy Market]]:** Armazenamento de manuais, faturas e ativos digitais com partilha segura via [[Goy ID]].
*   **Goy Mail:** Gestão de anexos como links temporários e cifrados para otimização de espaço.

## 3. Funcionalidades de Vanguarda
*   **Goy Vault:** Pasta ultra-segura que requer assinatura multi-dispositivo para acesso.
*   **Partilha Granular via NPUB:** Acesso restrito a utilizadores específicos, onde apenas a chave privada do destinatário consegue decifrar o ficheiro.
*   **Streaming Nativo:** Descodificação em tempo real para vídeos e áudio diretamente no browser ou app.

## 4. Monetização (Máquina de Cash Flow)
Lucro baseado em infraestrutura e serviços premium:
1.  **Modelo Freemium:** 5GB gratuitos com planos expansíveis pagos em Sats via [[Goy Pay]].
2.  **Goy Cloud for Business:** Relays dedicados com redundância geográfica e suporte corporativo.
3.  **Cold Storage:** Backups de baixo custo (estilo Glacier) geridos via Lightning.
4.  **B2B SDK:** APIs para que outros programadores utilizem a infraestrutura de armazenamento da Goy Co.
